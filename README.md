# Análise Combinatória (Java WEB)

Projeto criado para automatizar os cálculos na parte de análise combinatória.

Projeto criado no NetBeans utilizando a linguagem Java voltada para WEB a partir das versões desenvolvidas em [C#](https://github.com/iagocolodetti/Analise_Combinatoria "https://github.com/iagocolodetti/Analise_Combinatoria") e [Java](https://github.com/iagocolodetti/Analise_Combinatoria_Android "https://github.com/iagocolodetti/Analise_Combinatoria_Android") para Android.

Aplicação criada com o padrão MVC, utilizando classe Java como modelo, páginas JSP como view (com Bootstrap 4 e JSTL) e Servlets como controller.
